Pour tester l'app il faut aller dans home et refresh a chaque modification
Toutes les fonctionnalités sont dans le dashboard

Étapes pour lancer l'app :


1. cd Food-App
2. cd server
3. npm install firebase-tools --save-dev
4. npx firebase-tools --version
5. cd functions
6. npm install
7. npm run serve (#Pour enfin run le server)


Prochaine Étapes : 

1. Ouvrir un autre terminale
2. cd Food-App
3. cd Client
4. supprimer le fichier yarn.lock
5. yarn install
6. yarn start (#Pour enfin run l'app)
